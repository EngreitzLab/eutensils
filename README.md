# eutensils

Shared **R & Python** utilities for the Engreitz Lab — handy tools for single-cell, CRISPR/TAP-seq, analysis workflows, plotting, QC, and I/O.

> Reusable, lab-tested bits and pieces — so we don’t keep rewriting the same code.

---

## Features (initial scope)
- **Single-cell helpers:** Seurat/Scanpy glue, metadata joins, plotting themes
- **CRISPR/TAP-seq utils:** primer checks, guide labeling, efficiency summaries
- **I/O & munging:** fast readers/writers, Google Cloud helpers, schema checks
- **Plotting:** consistent themes, palettes, small multiples for QC
- **Repro:** tiny test datasets and deterministic examples

*(Add/remove as the repo evolves.)*
